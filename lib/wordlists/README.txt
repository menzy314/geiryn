This directory contains wordlists extracted from Lecsicon Cymraeg Bangor
https://github.com/techiaith/lecsicon-cymraeg-bangor

makelists.js - script to extract five-letter wordforms from lecsicon_cc0.txt
geiryn.cyLonglist.js - five-letter wordforms
geiryn.cyShortlist.js - five-letter lemmas/plurals. Needs reducing by hand to frequent words
